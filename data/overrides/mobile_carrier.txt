# mobile_carrier.txt — ASNs of DEDICATED cellular carrier networks.
#
# Effect: sets flags bit 10 (mobile_carrier). Classification: :mobile
# (precedence rule 11). :mobile is a likely_human verdict; its value is
# telling apps "one IP here = hundreds of humans behind carrier NAT — never
# hard-block, never per-IP rate-limit aggressively".
#
# Curation bar (deliberately strict for the seed):
#   * The ASN carries a carrier's MOBILE subscriber traffic specifically —
#     not a converged telco ASN mixing fixed broadband on the same AS
#     (those stay :residential_isp, which is equally likely_human, so a
#     miss here costs almost nothing).
#   * Fixed-wireless ISPs (WISPs) do NOT belong here: their subscribers are
#     homes with stable IPs — that's :residential_isp, not carrier NAT.
#   * Known incompleteness is fine; recall grows via PRs. Misclassified
#     entries are the only real bug class for this file.
